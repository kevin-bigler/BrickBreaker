BrickBreak

This Game / Library is my way of learning code and will eventually become a full featured Library to help other beginners, Learn by making a simple game like brick breaker! --Game Rules--

  • You start with 3 lives
  • if ball reaches the bottom of the screen you lose a life
  • press enter to start the game
  • special colored bricks give power ups!
  • beat as many levels as you can !

    more powerups to come soon!
